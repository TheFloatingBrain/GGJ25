# What is it?
 - Mini 3D Game engine with Physics
 - Physics Demo
 - Character Demo

# How To?

## Linux
Clone the repo, then
```
git clone ...
cd GGJ25
mkdir build
cd build
cmake ../
make -j$(nproc)
```

## Windows
Clone the repo, then
```
git clone ...
cd GGJ25
mkdir build
cd build
cmake ../
msbuild ALL_BUILD.vcxproj
```
